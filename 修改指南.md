# 📝 HTML PPT 模板修改指南

## 🎯 三种修改方式(从易到难)

---

## 方式一:使用文本编辑器(最简单) ✅ 推荐新手

### 步骤:
1. **右键点击HTML文件** → 选择"打开方式" → 选择"文本编辑"(Mac)或"记事本"(Windows)
2. **推荐使用更好的编辑器**:
   - Mac: TextEdit、VS Code、Sublime Text
   - Windows: Notepad++、VS Code
   - 跨平台: VS Code(免费,最推荐)

3. **找到要修改的文字**,直接替换
4. **保存文件**(Cmd+S 或 Ctrl+S)
5. **双击HTML文件**在浏览器中查看效果

### 示例:
```html
<!-- 修改前 -->
<h1>我的日常记录</h1>

<!-- 修改后 -->
<h1>张三的工作日记</h1>
```

---

## 方式二:使用浏览器开发者工具(实时预览)

### 步骤:
1. **双击打开HTML文件**在浏览器中
2. **右键点击要修改的文字** → 选择"检查"或"审查元素"
3. **在右侧代码区直接修改文字**
4. **看到满意效果后**,复制修改后的代码回HTML文件

### 优点:
- ✅ 实时看到效果
- ✅ 不用担心改坏代码结构

### 缺点:
- ❌ 刷新页面后修改会丢失
- ❌ 需要手动把修改复制回原文件

---

## 方式三:在线编辑器(无需安装软件)

### 推荐工具:
1. **CodePen** (codepen.io)
2. **JSFiddle** (jsfiddle.net)
3. **GitHub Editor** (github.com)

### 步骤:
1. 打开任意一个在线编辑器
2. 复制HTML文件内容粘贴进去
3. 修改内容
4. 下载或复制修改后的代码

## 🔍 具体修改位置说明

### 1️⃣ 修改标题和文字

在HTML文件中查找这些标记:

```html
<!-- 主标题 -->
<h1>这里的内容可以改</h1>

<!-- 副标题 -->
<p class="subtitle">这里也可以改</p>

<!-- 普通文字 -->
<div class="todo-text">待办事项内容</div>
```

### 2️⃣ 修改日期

```html
<!-- 找到这一行,修改日期 -->
<div style="font-size: 1.3em; margin-bottom: 30px; opacity: 0.8;">
    2024年12月15日 星期日  <!-- 👈 改成今天的日期 -->
</div>
```

### 3️⃣ 添加/删除待办事项

```html
<!-- 复制这个块来添加新事项 -->
<div class="todo-item">
    <div class="todo-checkbox"></div>
    <div class="todo-text">你的新事项</div>
</div>

<!-- 删除整个 div 块就可以移除事项 -->
```

### 4️⃣ 修改颜色主题

在 `<style>` 部分找到 `background:` 属性:

```css
.slide-1 {
    /* 原来是 */
    background: linear-gradient(135deg, #ffecd2 0%, #fcb69f 100%);
    
    /* 可以改成其他颜色组合 */
    background: linear-gradient(135deg, #a8edea 0%, #fed6e3 100%);
}
```

### 常用颜色代码:
- 温暖橙色: `#ffecd2` → `#fcb69f`
- 清新绿色: `#e8f5e9` → `#c8e6c9`
- 温柔粉色: `#ffeaa7` → `#fab1a0`
- 宁静蓝色: `#a8edea` → `#fed6e3`
- 优雅紫色: `#dfe6e9` → `#b2bec3`

## 💡 实用技巧

### ✅ 安全修改原则:
1. **只修改引号内的文字**,不要动标签本身
2. **备份原文件**后再修改(复制一份副本)
3. **小步修改**,改一点就保存查看效果
4. **不要删除** `<`, `>`, `/`, `=` 这些符号

### ❌ 避免这些操作:
1. 不要删除整个 `<div>` 或 `<style>` 标签
2. 不要修改 `class=` 后面的类名
3. 不要删除中文注释(`<!-- ... -->`)外的特殊符号

## 🆘 遇到问题怎么办?

### 问题1: 修改后显示乱码
**解决**: 确保文件保存为 UTF-8 编码

### 问题2: 不小心改坏了
**解决**: 
1. 用备份文件恢复
2. 或者重新下载原始模板

### 问题3: 不知道某段代码是干什么的
**解决**: 
1. 先不要动它
2. 或者截图问我

### 问题4: 想添加图片
**解决**: 需要添加 `<img>` 标签,我可以帮你修改
